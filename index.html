Uncertainty Explanation

Baseline Paper Code Baselie Code Dataset

Abstract

Robotic agents equipped with large language models (LLMs) have the potential to enhance human-robot interaction by understanding and reasoning about user commands. There is active research on embracing LLMs into physical robots or utilizing LLMs as planners with their reasoning capabilities. On the other hand, as the user's command can be ambiguous or even infeasible, proper interpretation of user commands can be an essential component for achieving the reliability of the interactive system. In this project, we aim to generate an explanation of the uncertainty given uncertain commands.
